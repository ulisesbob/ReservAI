## SEO Technical Requirements

### On-Page SEO
- H1: Target main keyword only
- H2: Related keywords
- Meta description: 155 characters, include main keyword
- URL slug: descriptive, hyphens, 50 chars max
- Internal links: 3-5 per post to other posts/pages
- External links: 2-3 authority sources
- Images: Alt text with keyword (naturally)
- Word count: 1200-2000 words (longer = better ranking for competitive keywords)

### Content Structure
- Introduction (hook + keyword mention)
- Section 1-4 with H2s (clear structure)
- List/bullets when appropriate (scannable)
- Conclusion with CTA
- Author box (Ulises)
- Related posts section (internal linking)

### Technical SEO
- Site speed: <3 seconds page load
- Mobile responsive: 100/100 Google Mobile Friendly
- SSL: HTTPS only
- XML sitemap: Include blog URLs
- Robots.txt: Allow all blog content
- Structured data: Schema.org for Article type

### Link Building
- Guest posts (1-2/month on food/biz blogs)
- Mention in industry reports (reach out)
- Forum answers (Reddit, Quora, LinkedIn)
- Social shares (drive traffic + signals)
- Directory listings (local directories)
